Exchange Rate Volatility Modeling

Quantifying market risk using GARCH-family models

Problem Statement

Traditional time-series models assume constant variance, which is unrealistic for financial data such as exchange rates. This project seeks to model time-varying volatility in Nigeria’s exchange rate and determine whether GARCH models adequately capture observed volatility patterns.

Data Description

The dataset consists of time-series exchange rate data obtained from the Central Bank of Nigeria (CBN). The data records periodic exchange rate values over time and is suitable for volatility and financial risk analysis.

Data Preparation

  • Converted exchange rate series into returns.
  • Conducted descriptive statistical analysis.
  • Checked for stationarity using unit root tests.
  • Tested for the presence of ARCH effects.

Methodology

The analysis workflow involved:

  • Computation of log returns.
  • Stationarity testing (ADF test).
  • Estimation of ARCH and GARCH models.
  • Model diagnostics and residual analysis.

Analysis & Visualizations

1. Exchange Rate Return Series (Line Plot)

The plot illustrates fluctuations in exchange rate returns over time, showing periods of high and low volatility.

2. Volatility Clustering (Conditional Variance Plot)

The conditional variance plot reveals clear volatility clustering, where periods of high volatility are followed by similar periods.

3. GARCH Model Diagnostics (Residual & ACF Plots)

Diagnostic plots confirm that the GARCH model adequately captures time-varying volatility, with reduced autocorrelation in standardized residuals.

Conclusion

This project demonstrates the application of GARCH models in analyzing financial time-series data. The results confirm that exchange rate volatility is time-varying and clustered, highlighting the importance of advanced econometric models in financial risk analysis and policy evaluation.

Tools Used

R EViews Time Series Analysis

Key Findings

✅ Exchange rate returns exhibit volatility clustering.

✅ Presence of significant ARCH effects in the data.

✅ GARCH models effectively capture time-varying volatility.

✅ Variance persistence indicates long-memory behavior in exchange rate volatility.
